## ⚠️ Read this first / Lee esto primero

- **Back up your save** before using NHLE. Editing memory can corrupt your island.
- **Never use it online.** Disconnect from the internet on the Switch and play offline only.
- NHLE only works with **Animal Crossing: New Horizons version 3.0.3**.
- This tool is **not affiliated with Nintendo**.
